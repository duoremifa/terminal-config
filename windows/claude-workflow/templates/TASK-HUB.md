# 📋 任务中心

> 所有项目的任务汇总在这里。任务分散在各个项目里，但在这里统一看。
> 依赖 Obsidian 插件：**Tasks**（查询）+ **Full Calendar**（日历）

---

## 🔥 今天到期

```tasks
due before tomorrow
not done
sort by due
group by filename
```

---

## 📅 本周到期

```tasks
due after today
due before next week
not done
sort by due
group by filename
```

---

## 📝 待安排（无日期）

```tasks
no due date
not done
sort by filename
group by filename
```

---

## ✅ 最近完成（7 天内）

```tasks
done
done after 7 days ago
sort by done reverse
group by filename
```

---

## 📆 看日历

按 `Ctrl + P`，输入 `Open Full Calendar`，回车。

带 `📅` 日期的任务会自动显示在日历上。

---

## 📖 任务格式速查

在任何项目的 `.md` 文件里这样写任务：

```markdown
- [ ] 任务描述 📅 2026-07-05     # 到期日，显示在日历
- [ ] 任务描述 ⏳ 2026-07-03     # 计划日
- [ ] 任务描述                   # 无日期，在"待安排"里
- [x] 已完成的任务 ✅ 2026-07-01  # 完成
```

标记完成：把 `- [ ]` 改成 `- [x]`，或在任务行按 `Ctrl+Enter`。
