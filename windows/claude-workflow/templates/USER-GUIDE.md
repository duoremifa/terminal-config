# 📘 终端 + Claude 操作指南

> 自用 / 给同事的操作指南。忘了命令就翻这份。

---

## 一、核心概念

- **终端** = 黑底白字窗口（Windows Terminal / iTerm2 / Terminal.app）
- **项目** = 项目根目录下的每一个文件夹
- **会话** = 你和 Claude 的一次对话，按项目独立保存

---

## 二、命令速查

| 命令 | 干嘛的 |
|---|---|
| `pl` | 列出所有项目 |
| `pj` | 看菜单选项目进入 |
| `pj 项目名` | 直接进入指定项目 |
| `cc` | 继续当前项目上次会话 |
| `cr` | 从历史里挑一个会话恢复 |
| `Ctrl+C` (两次) | 退出 Claude |
| `/projects` | 在 Claude 内切换项目 |
| `/exit` | 退出 Claude |

---

## 三、典型一天

```powershell
# 早上开工
pj              # 列菜单选项目

# 干活
# ...（和 Claude 聊、写文档、处理数据...）

# 切项目
pj 另一个项目    # 或 /projects

# 接昨天进度
cc

# 翻历史
cr
```

---

## 四、任务管理

在任何 `.md` 文件里写任务：

```markdown
- [ ] 任务描述 📅 2026-07-05
- [ ] 任务描述
```

- `📅` = 到期日，显示在日历
- 不带日期 = 待安排

**看任务**：Obsidian 打开 `任务中心.md`

**看日历**：`Ctrl+P` → `Open Full Calendar`

**让 Claude 帮你管理任务**：直接说"帮我建个任务：xxx，下周五到期"

---

## 五、Obsidian 使用

- 打开 vault = 项目根目录
- 左侧栏能看到所有项目文件夹
- 每个项目根目录有 `CLAUDE.md`（项目说明）和 `TODO.md`（任务）
- 所有 `.md` 文件都能直接看、直接改

---

## 六、常见问题

**Q: 命令没反应？**
A: 关掉终端，重开一个新的。

**Q: Claude 卡住？**
A: 按 `Ctrl + C` 中断，重新输入问题。

**Q: 想建新项目？**
A: 在项目根目录下新建文件夹就行。`pj 新项目名` 会自动建。

**Q: Obsidian 看不到新文件？**
A: `Ctrl+P` → `Reload app without saving`
